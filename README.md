# homebrew-kolt

Homebrew tap for [kolt](https://github.com/snicmakino/kolt), a lightweight build tool for Kotlin.

## Install

```sh
brew install snicmakino/kolt/kolt
```

macOS arm64 (Apple Silicon) only. On other platforms use the [installer script](https://github.com/snicmakino/kolt#installation).

The formula's version and sha256 are bumped automatically by kolt's release workflow — do not edit them by hand.
